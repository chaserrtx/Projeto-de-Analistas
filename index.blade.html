PoliData

Sistema de Perfis de Analistas Políticos

Plataforma para gerenciamento de usuários, especialistas e perfis voltados para análise política, comunicação pública e monitoramento institucional.

Objetivos do Sistema

Cadastro de Usuários

Gerenciamento de usuários com nome, e-mail e senha.

Perfis Profissionais

Registro de biografia, telefone e informações profissionais.

Organização de Dados

Estrutura baseada em banco de dados relacional utilizando Laravel.
